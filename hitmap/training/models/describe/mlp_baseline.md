# MLPBaseline Model

## 모델 개요
가장 기본적인 다층 퍼셉트론(MLP) 구조의 베이스라인 모델로, 모든 입력 정보를 단일 벡터로 결합하여 처리합니다.

## 네트워크 구조
1. **Fully Connected Layers**:
   - `Linear` -> `BatchNorm1d` -> `ReLU`가 반복되는 구조.
   - 기본 은닉층 구성: [512, 512, 256, 128].
   - 배치 정규화(Batch Normalization)를 통해 학습 안정성을 확보합니다.

2. **Output Layer**:
   - `Linear(128, out_dim)`: 최종 4개 값([x, y, z, Fz])을 회귀.

## 입력 및 출력
- **입력**: 16개 센서 신호 + 1개 반지름 = 17차원 벡터.
- **출력**: 4차원 벡터 ([x, y, z, Fz]).

## 학습 방식
- 공간적인 구조(4x4)를 무시하고 1차원으로 평탄화하여 학습합니다.
- 복잡한 구조의 모델들과 성능을 비교하기 위한 기준점(Baseline) 역할을 합니다.
