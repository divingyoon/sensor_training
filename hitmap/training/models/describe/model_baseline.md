# BaselineModel (Phase 1)

## 모델 개요
Phase 1에서 사용된 모델로, MLP 인코더와 CNN 디코더를 결합하여 촉각 신호로부터 고해상도 압력 맵(Heatmap)을 생성합니다.

## 네트워크 구조
1. **MLP Encoder**:
   - 14채널 촉각 신호를 입력받아 128차원의 잠재 벡터(Latent Vector)를 생성.

2. **Aux Encoder**:
   - 보조 정보(Fx, Fy, z, radius)를 32차원으로 인코딩.

3. **Feature Fusion & Reshape**:
   - 촉각 잠재 벡터와 보조 벡터를 결합(160차원)한 후, 512차원으로 확장하여 (B, 8, 8, 8) 형태의 공간 특징 맵으로 변환.

4. **CNN Decoder (Upsampling)**:
   - 3단계의 `ConvTranspose2d`를 통해 8x8 공간 특징을 64x64 해상도로 확대.
   - 마지막 레이어에서 `Sigmoid`를 사용하여 0~1 사이의 압력 맵 생성.

## 입력 및 출력
- **입력**: 
  - `tactile`: (B, 14) 가용 센서 신호.
  - `aux`: (B, 4) 보조 물리 정보.
- **출력**: `hr_map`: (B, 1, 64, 64) 고해상도 촉각 이미지.

## 학습 방식
- 실제 센서 데이터와 목표 고해상도 맵 간의 차이를 픽셀 단위로 학습합니다.
- 물리적인 보조 정보를 강하게 결합하여 맵 생성의 정확도를 보조합니다.
