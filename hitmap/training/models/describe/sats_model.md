# SATS (Self-Attention-Assisted Tactile SR) Model

## 모델 개요
각 개별 센서의 시계열 특징을 먼저 추출한 뒤, Self-Attention 메커니즘을 통해 센서 간의 전역적인 공간 상관관계를 학습하는 모델입니다.

## 네트워크 구조
1. **Temporal Encoder (Per-sensor LSTM)**:
   - 16개 센서 각각에 대해 독립적으로 LSTM을 적용하여 시계열 특징(64차원)을 추출.
   - 각 센서가 겪은 압력 변화의 이력을 개별적으로 인코딩합니다.

2. **Self-Attention Module**:
   - 추출된 16개 센서의 특징들 간에 Attention 연산을 수행.
   - 센서 배열의 비대칭성이나 특정 센서 간의 전역적인 관계를 동적으로 가중치화합니다.

3. **MLP Decoder**:
   - Attention 출력(16 * 64)을 입력받아 고차원 추론을 수행.
   - 은닉층: [512, 256].

## 입력 및 출력
- **입력**: (B, T, 16) - 16개 센서의 시간 시퀀스.
- **출력**: (B, 4) - [x, y, z, Fz].

## 학습 방식
- 개별 센서의 시계열 문맥을 먼저 파악한 후, 센서 간의 공간적 관계를 합치는 방식을 취합니다.
- 특정 센서가 고장나거나 이상치가 발생했을 때 Attention을 통해 이를 보완하는 능력이 뛰어납니다.
