# IsolineGNN Model

## 모델 개요
Isoline(등압선) 이론에 기반하여 센서 간의 물리적 상호작용을 학습하기 위해 설계된 모델입니다. 그래프 구조와 유사한 방식으로 공간적인 특징을 처리합니다.

## 네트워크 구조
1. **Spatial Interact (Node-to-Edge Processing)**:
   - `Linear(16, 128)` + `ReLU` + `Linear(128, 128)` + `ReLU`.
   - 16개 센서 신호 간의 전역적인 상호작용을 임베딩하여 공간적 특징을 생성합니다.

2. **FC Head**:
   - 입력: 상호작용 특징(128) + 반지름(1).
   - `Linear(129, 256)` + `ReLU` + `Linear(256, 128)` + `ReLU` + `Linear(128, out_dim)`.

## 입력 및 출력
- **입력**:
  - `s16`: (B, 16) - 16채널 센서 신호.
  - `radius`: (B, 1) - 인덴터 반지름.
- **출력**: `(B, out_dim)` (일반적으로 4: [x, y, z, Fz]).

## 학습 방식
- 단순한 그리드 구조를 넘어 센서 신호 간의 상관관계를 밀집(Dense) 연결된 레이어를 통해 학습합니다.
- 등압선 이론에 따른 압력 분포의 변화를 고차원 특징 공간에서 모델링하여 위치 및 힘을 추정합니다.
