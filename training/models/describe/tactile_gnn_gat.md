# TactileGAT Model

## 모델 개요
Graph Attention Network(GAT)를 촉각 센서에 적용한 모델입니다. 센서들을 그래프의 노드로 정의하고, 각 센서 쌍 사이의 관계 가중치를 동적으로 계산하여 특징을 집계합니다.

## 네트워크 구조
1. **GAT Layer**:
   - 16개 센서를 완전히 연결된(Fully Connected) 그래프로 간주.
   - **Linear Projection**: 각 노드의 신호를 고차원(32) 특징으로 변환.
   - **Attention Mechanism**: 각 센서 쌍(i, j)의 특징을 결합한 후 학습 가능한 벡터 `a`와 내적하여 점수 산출.
   - **Aggregation**: Softmax로 정규화된 점수를 가중치로 하여 이웃 센서들의 정보를 통합.
   - Multi-head Attention(2 heads)을 사용하여 다양한 관계를 동시에 학습.

2. **FC Regressor**:
   - 통합된 16개 센서의 특징을 입력받아 최종 회귀 수행.

## 입력 및 출력
- **입력**: (B, 16) - 16채널 센서 신호.
- **출력**: (B, 6) - [x, y, z, Fx, Fy, Fz].

## 학습 방식
- 센서 간의 기하학적 거리에 국한되지 않고, 데이터로부터 유의미한 상호작용 관계를 직접 학습합니다.
- 특정 지점에 압력이 가해졌을 때 영향을 받는 센서 그룹 간의 의존성을 효과적으로 모델링합니다.
