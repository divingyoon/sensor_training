# 16채널 촉각 지능 아키텍처 학습 상세 보고서 (2026-04-06)

본 보고서는 하드웨어(6.5mm 간격 센서 어레이)의 물리적 해상도를 소프트웨어적으로 확장하여 **0.3mm급 위치 정밀도**와 **0.6N급 힘 추정**을 달성한 학습 구조 및 최적화 전략을 상세히 기술합니다.

---

## 1. 모델 아키텍처 및 데이터 흐름 (System Architecture)

본 시스템은 두 개의 직렬 연결된 MLP(Multi-Layer Perceptron) 네트워크로 구성됩니다.

### 1.1 Super Resolution (SR) Model: 위치 및 깊이 추론
*   **Input (17-dim)**: 16채널 정규화 신호 ($s_{norm}$) + 인덴터 반지름 ($r$)
*   **Structure**: 
    - Input Layer (17) → Linear(512) → BN → ReLU → Dropout(0.1)
    - Linear(512) → BN → ReLU → Dropout(0.1)
    - Linear(256) → BN → ReLU → Dropout(0.05)
    - Linear(128) → BN → ReLU
    - Output Layer (3): [Normalized $x$, $y$, $z_{depth}$]
*   **특징**: 심층 네트워크를 통해 센서 간 비선형 간섭 패턴을 고차원에서 매핑하여 서브 밀리미터(Sub-mm) 단위의 미세 좌표를 추출합니다.

### 1.2 Force Field (FF) Model: 수직 항력 추론
*   **Input (20-dim)**: 16채널 신호 + 반지름 ($r$) + **SR 결과값 (x, y, z_depth)**
*   **Structure**: 
    - SR에서 추론된 정규화 좌표를 입력값의 일부로 재주입(Input Concatenation)하여, "어디서(Position)" "얼마나 깊게(Depth)" 눌렸는지의 기하학적 정보를 힘 추정에 직접 활용합니다.
    - 4-Layer MLP 구조 (512-512-256-128)
*   **Output (1-dim)**: Normalized $Fz$ (수직 항력)

## 2. 핵심 최적화 매커니즘 (Technical Deep Dive)

### 2.1 시공간 동기화 및 정지 상태 필터링 (Stationary Filtering)
*   **물리적 배경**: 실험용 3축 스테이지의 좌표 갱신 속도와 센서의 샘플링 속도 간 약 200ms의 시간 지연(Time-lag) 존재. 이동 중 데이터를 학습할 경우, X축 이동 방향에 따라 실제 접촉 위치와 데이터 좌표가 **±1.01mm (총 2.02mm)** 어긋나는 히스테리시스 발생.
*   **해결책**: 데이터셋 전처리 시 $dx/dt$ 및 $dy/dt$가 0에 수렴하는 **정지 상태(Stationary) 데이터**만 100% 추출. 이를 통해 센서 반응과 실제 좌표 간의 1:1 물리적 대응성을 확보하여 원천적으로 바이어스 제거.

### 2.2 하이브리드 타겟 정규화 (Target Normalization & Scaling)
단순한 Min-Max Scaling을 넘어, 모델이 학습 시 모든 출력축에 대해 동일한 Gradient Scale을 가지도록 정규화 파라미터를 관리합니다.

*   **정규화 파라미터 관리**:
    - $X, Y \in [-9.75, 9.75] \rightarrow [0, 1]$
    - $Z \in [0, 1.5] \rightarrow [0, 1]$
    - $Fz \in [-1.26, 2.77] \rightarrow [0, 1]$
*   **체크포인트 통합**: 모델 저장 시 해당 학습 데이터셋의 `target_norm` 파라미터를 `.pt` 파일 내에 함께 저장하여, 추론 시 별도의 설정 없이도 실제 물리 단위(mm, N)로 즉시 환산 가능하도록 설계.

### 2.3 기하학적 왜곡 보정 (Weighted X-axis Loss)
*   **문제**: 센서 어레이 배치 특성상 X축 방향의 센서 간격이 Y축보다 불균일하여, X축 방향의 공간 정보 손실이 더 큼.
*   **해결책**: 손실 함수에 가중치 계수($W$) 적용.
    $$Loss = W_x \cdot MSE(x) + W_y \cdot MSE(y) + W_z \cdot MSE(z)$$
    여기서 $W_x=2.0 \sim 3.0$을 적용하여 모델이 X축 오차 감소에 더 많은 에너지를 쏟도록 유도.

## 3. 학습 성능 상세 결과

### 3.1 SR 모델 정밀도 (Total MAE: 0.338 mm)
*   **X축 (0.62mm)**: 하드웨어 배치 한계로 인해 가장자리 오차가 존재하나, 가중치 학습 전(1.2mm) 대비 **약 50% 개선**.
*   **Y축 (0.31mm)**: 센서 배치가 조밀한 방향으로, 매우 안정적인 선형성을 보임.
*   **Z축 (0.08mm)**: 기압 기반 센서의 특성상 압입 깊이에 매우 민감하여, 머리카락 두께 수준의 미세 변위 감지 가능.

### 3.2 FF 모델 정밀도 (Fz MAE: 0.602 N)
*   SR 모델이 제공한 고정밀 위치 정보를 바탕으로, 동일한 신호 세기라도 "중앙에서 눌렸을 때"와 "가장자리에서 눌렸을 때"의 압력 차이를 학습하여 **비선형적 힘-신호 관계**를 정확히 모델링함.

## 4. 프레임워크 확장성 및 실무 활용

본 프레임워크는 다음과 같은 실무적 이점을 제공합니다.

1.  **고속 모델 업데이트**: VRAM Caching 기술을 통해 신규 데이터 추가 시 5분 이내에 재학습 완료 가능.
2.  **데이터 품질 진단**: `visualize_grid_errors.py` 도구를 통해 센서 어레이 중 물리적으로 손상되거나 특성이 변한 지점을 히트맵으로 즉시 식별 가능.
3.  **실시간성**: 단순 MLP 구조를 채택함으로써, 엣지 컴퓨팅 환경에서도 1kHz 이상의 속도로 SR 및 FF 동시 추론 가능.
