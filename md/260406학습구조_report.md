3D 센서 SR 및 힘 추정 프레임워크 분석과 제안
현 프레임워크 요약
현재 시스템은 16채널 촉각 센서 배열을 대상으로 0.3 mm 수준의 위치 정밀도와 0.6 N 수준의 힘 추정 정확도를 달성하기 위해 두 개의 연속된 MLP 구조를 사용하고 있다. 첫 번째 모델(SR 모델)은 16채널 신호와 인덴터 반지름을 받아 정규화된 , ,  깊이를 출력하며, 두 번째 모델(FF 모델)은 SR 결과와 원본 신호를 합쳐 수직 힘 를 추정한다. 정지 상태 데이터만을 학습에 사용하고, 축별 손실 가중치 및 일관된 정규화를 통해 좌표 오차를 줄였다. 이러한 구조로 Y축과 Z축의 평균 절대 오차(MAE)는 각각 0.31 mm, 0.08 mm 수준에 도달했지만 X축 오차는 0.62 mm로 여전히 상대적으로 크다. 또한 물리적 센서 배치의 불균형 때문에 위치별로 X·Y 오차 특성이 다르게 나타난다.
그리드별 오차 분석
1. 축별 오차 불균형 원인
    1. 센서 간 간격의 비대칭성: 현재 센서 어레이는 X축 간격이 Y축 간격보다 넓어 수평() 방향의 분해능이 낮다. 물리적 배치가 불균형하면 동일한 입력 변화에 대해 센서 출력의 기울기가 다르므로 MLP가 학습할 때 X축 정보의 상대적 중요도를 놓치기 쉽다. 보고서에서도 X축 손실 가중치 를 사용하여 이를 보상하였다.
    2. 커버 구조의 비선형 응답: 센서 위 커버 소재(실리콘 등)는 위치에 따라 탄성 변형이 달라지고, 측면의 응력 전달 경로가 복잡하다. 특히 가장자리에서는 압력이 비대칭으로 분산되어 특정 축만 큰 오차가 발생할 수 있다. 이런 현상은 논문에서 설명된 것처럼 센서의 겹치는 수용영역을 통해 주변 자극을 합성할 때 생기는 잡음과 유사하다[1].
    3. 데이터 수집 히스테리시스: 실험 장비의 이동과 센서 샘플링 사이에 시간 지연이 있어, 이동 중에 수집된 데이터는 실제 접촉 위치와 mismatched 될 수 있다. 보고서에서 정지 상태 데이터를 선택한 이유도 이 문제를 회피하기 위함이다. 그러나 공기 압력 기반 센서의 히스테리시스는 완전히 제거되지 않아 X, Y 축의 일부 위치에서 오차가 증가할 수 있다.
    4. 학습 모델의 표현력 제한: 단순 MLP는 공간적 인접성을 명시적으로 표현하지 못한다. 따라서 특정 위치에서의 오차 패턴을 글로벌한 가중치로 보상하게 되고, 위치별 미세한 비선형성을 포착하지 못한다. 이는 X축과 Y축에서 서로 다른 위치 패턴을 보이는 직접적인 원인이 된다.
고차원 SR 및 힘 추정을 위한 모델링 전략
기존 MLP 기반 접근은 가볍고 빠르지만, 복잡한 비선형 현상을 충분히 포착하지 못한다. 보다 향상된 3D SR() 및 힘장(force field) 추정을 위해서는 공간적 패턴을 다루는 모델이 필요하다. 최근 연구들은 LSTM, Self‑Attention, CNN 및 변분 오토인코더를 결합하여 촉각 SR을 달성했다. 몇 가지 대표적 방법과 적용 가능성을 분석한다.
1. Self‑Attention 기반 SATS 모델
    • 구조: 과학 Advances 논문에서 제안된 Self‑Attention–Assisted Tactile SR(SATS) 모델은 센서 배열을 그래프로 간주하고 각 센싱 유닛을 노드로 표현한다. LSTM 인코더가 각 센서의 시계열 특성을 추출한 후, self‑attention 모듈이 인접 노드들의 정보를 통합하여 공간적 상호작용을 학습한다. 이어서 MLP 디코더와 CNN refining 모듈이 고해상도 맵을 생성한다[2]. 이 모델은 23개의 실제 센서로부터 2 700개의 가상 택셀을 생성하며 평균 위치 오차 0.73 mm를 달성했다[3].
    • 적용 방안: 현재 시스템의 16채널 신호를 그래프 구조로 표현하고 각 센서별 LSTM 인코더를 두어 시계열 감쇠를 학습한다. self‑attention 모듈을 통해 이웃 센서의 반응을 통합하면 X·Y 축의 비대칭 문제를 줄일 수 있다. 마지막에 2D CNN을 사용해 위치별  맵을 구축하고 FF 모델과 결합해 수직 힘 맵을 출력하는 구조가 가능하다.
2. VAE + Residual Upsampling 기반 SuperTac
    • 구조: SuperTac 프레임워크는 저해상도 촉각 맵을 Variational Autoencoder(VAE)로 압축·표현한 뒤, Residual‑In‑Residual Blocks(RIRB)를 통해 업샘플링하여 4배 이상의 공간 해상도를 달성한다[4]. VAE는 센서 출력의 잠재 구조를 학습하여 잡음을 제거하고, RIRB는 고주파 공간 특징을 복원한다. 텍스처 분류에서 원본보다 17% 정확도가 높아졌다[5].
    • 적용 방안: 16채널 입력을 4×4 맵 형태로 재구성한 후, VAE로 저차 잠재 벡터를 추정하고 RIRB 기반 업샘플러로 32×32 혹은 64×64 해상도로 확장한다. 결과 맵에서 가장 높은 압력 위치를 추출하여  좌표를 결정하고 z‑depth와 힘값을 함께 회귀하는 방식으로 사용할 수 있다.
3. CNN + LSTM 하이브리드 모델
    • 개념: CNN은 센서 배열에서 공간 패턴을 추출하는 데 강력하며, LSTM은 시계열 히스테리시스를 처리한다. CoP(중심압) 추정 연구에서는 CNN-LSTM, Bi‑LSTM 등 여러 모델을 비교한 결과 CNN-Bi‑LSTM이 가장 낮은 오차를 보였다는 보고가 있다. CNN을 통해 2D 압력 이미지를 특징맵으로 변환하고, LSTM이 시간적 변화를 처리하면 동적 접촉에서 정밀도가 향상된다.
    • 적용 방안: 16채널 값을 4×4 행렬로 표현하고 여러 시점(예: 몇 개의 샘플 윈도우)으로 스택하여 3D 텐서로 만든다. 이후 2D CNN으로 공간 특징을 추출하고 LSTM/GRU로 시계열 정보를 통합하여 를 동시에 예측한다. 센서 데이터의 움직임 속도가 높지 않다면 TCN(Temporal Convolutional Network)을 대안으로 사용할 수 있다.
4. Transformer 기반 모델
    • 사례: Vision‑based tactile sensor 연구에서 TransForce 모델은 시퀀셜 이미지 변환과 Transformer 기반 리커런트 네트워크를 통해 힘을 추정하였다. 이 모델은 새로운 센서에서도 기존 데이터로부터 학습한 force predictor를 이식(transferring)하며, LSTM 기반 모델보다 정확도가 높았다[6]. Self‑attention은 장거리 공간 상호작용과 다양한 조명·마커 패턴 변화를 잘 처리한다.
    • 적용 방안: 16채널 시계열을 한 프레임의 벡터 대신 자가-어텐션 입력으로 처리하는 트랜스포머 인코더를 설계한다. 포지셔널 인코딩을 통해 센서 위치를 나타내고, 여러 인코더 레이어에서 센서 간 상관관계를 학습한다. 이어 회귀 헤드를 통해 를 추정한다. 트랜스포머는 충분한 데이터와 연산 자원이 필요하므로, 데이터 증강 및 규제를 적용해야 한다.
5. 그래프 신경망(GNN) 기반 모델
    • 배경: SATS에서와 같이 센서 배열은 자연스럽게 그래프 구조로 표현된다. 인접 행렬을 정의하고 Graph Convolutional Network (GCN) 혹은 Graph Attention Network (GAT)를 활용하여 각 노드의 특성을 집계하면 공간적 비선형성을 효과적으로 학습할 수 있다.
    • 적용 방안: 16개 센서를 그래프의 노드로 하고, 거리에 따라 가중치를 둔 간선을 연결한다. 각 노드의 시계열 신호를 입력 특성으로 삼아 GCN/GAT 레이어를 통과시키면 x–y–z–force 추정을 위한 전역 표현을 얻을 수 있다. GNN은 센서 위치 및 연결 구조를 명시적으로 모델에 반영할 수 있는 장점이 있다.
데이터 수집 및 학습 전략
    1. 정밀한 데이터 취합: 스테이지 이동을 0.1 μm 단위로 수행하는 동안, 각 그리드 포인트에서 충분한 정지 상태 샘플을 확보해야 한다. x·y·z 범위 전 영역을 균일하게 커버하도록 계획해 데이터 불균형을 줄인다.
    2. 공간적/시간적 증강: 대칭 이동, 회전, 미세 노이즈 추가 등으로 데이터셋을 확장해 모델의 일반화를 돕는다. 시계열 모델에서는 작은 시간 지연이나 잡음을 추가해 히스테리시스에 대한 강인성을 높일 수 있다.
    3. 교차 검증과 테스트 분리: 전체 데이터셋을 위치 기반으로 분할하여, 특정 위치 영역을 테스트 셋으로 완전히 제외하는 방식의 교차 검증을 수행한다. 이렇게 하면 새 위치에서의 일반화 성능을 평가할 수 있다.
    4. 정규화 및 타깃 스케일 관리: 보고서에서처럼 각 출력 채널의 범위와 단위를 명확히 정의하고, 학습 시 동일한 그래디언트 스케일을 유지한다. checkpoint에 정규화 파라미터를 저장해 배포 시 편의성을 높인다.
    5. 손실 함수 설계: 다중 출력 회귀 문제이므로 각 출력에 대한 오차의 중요도를 고려한 가중합 손실을 사용한다. 예를 들어 위치 오차와 힘 오차를 동시에 최소화하기 위해 가중치를 할당한다. 물리적 한계나 안전 범위를 벗어난 예측에 페널티를 주는 규제 항을 추가하는 것도 고려한다.
구현 권장 사항
    1. 모델 선택: 데이터 양이 충분하고 실시간성이 핵심이 아니라면 SATS와 같은 attention+CNN 구조가 가장 강력하다. 실시간성이 중요하거나 연산 자원이 제한된다면 CNN‑LSTM 또는 GNN 기반 경량 모델을 고려한다. 간단한 MLP는 베이스라인으로 유지한다.
    2. 코드 구조화: 프로젝트를 dataset, models, train, eval 패키지로 구분하여 관리한다. models 폴더에는 MLP, CNN-LSTM, Transformer, GNN 등 다양한 구조를 모듈화해 두고, 설정에 따라 교체 가능하도록 설계한다.
    3. 학습 및 평가 루프: PyTorch를 사용한다면 Lightning 또는 accelerate 프레임워크를 활용해 분산 학습과 로그 관리의 편의성을 높인다. 학습 시 Early Stopping과 Learning Rate Scheduler를 사용하여 과적합을 방지한다.
    4. 시각화 도구: visualize_grid_errors.py와 같은 스크립트를 확장하여 위치별 오차 히트맵, 예측‑실측 곡선, 혼돈 행렬 등을 자동 생성하도록 한다. 새 모델 도입 전후의 오차 분포를 비교할 수 있어야 한다.
    5. 배포: 학습된 모델과 함께 정규화 파라미터 및 모델 구성 정보를 JSON 또는 YAML 형태로 저장해 추론 파이프라인에서 로드하도록 한다. 임베디드 시스템에서의 속도를 고려해 ONNX 변환 후 추론 속도를 검증한다.
결론
단순 MLP 기반 구조는 구현이 용이하고 빠르지만, 센서의 비선형 응답과 공간적 상호작용을 충분히 학습하기에는 한계가 있다. 다양한 연구 결과는 LSTM, Self‑Attention, VAE, Residual Upsampling, GNN 등 공간·시간적 특징을 명시적으로 모델링하는 기법이 촉각 센서의 공간 해상도와 힘 추정 정확도를 크게 향상시킬 수 있음을 보여준다[2][4]. 따라서 향후 시스템은 센서 배열 구조를 반영한 CNN·LSTM·Transformer·GNN 기반 모델을 채택하고, 체계적인 데이터 수집과 손실 설계를 통해 3D SR 및 힘장을 동시에 추정하는 방향으로 발전해야 한다.
