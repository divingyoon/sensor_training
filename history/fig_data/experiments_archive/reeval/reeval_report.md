## 종합 결론 (map 품질 재평가 반영, 2026-07-20)

`map_quality.md`(위치·peak 상관·magnitude)와 종합하면:

1. **rel RMSE 저force 왜곡 확정** — target 압력≈0(저force)에서 분모 폭발. 계단식 xy0.5는 저force 정지 프레임이 압도적이라 rel이 부당하게 나빴다. 절대 rmse·map 품질로는 왜곡 없음.
2. **위치는 전 모델 정확** — loc median 0.5mm(그리드 1셀). rel이 높아도 SATS map의 위치 재현은 최상.
3. **xy0.5 d5 > xy1 d5** — peak 상관 0.962 vs 0.781, rel(공정) 0.153 vs 0.297. **xy0.5 정보량 이점 실재**. "xy0.5 나쁨"은 착시.
4. **d10 자체는 약하지 않음** — xy1 d10 peak 상관 0.976·loc 0.5mm(최상). xy0.5 d10만 낮은(0.491) 이유 = 그 홀드아웃(test3)이 저force+3rep. d10 열세는 특정 홀드아웃 아티팩트.
5. **유일 실제 편향** = xy0.5 d10 저force magnitude 과대(peak 비율 1.205). 위치와 분리된 저force d10 취득 커버리지 문제(β/크기입력이 다뤘던 것과 동일 뿌리).

### 판정 정정
- 과거 "d10 약함/xy0.5 열세" 판정은 **rel 저force 왜곡 + 저force 홀드아웃**의 조합 아티팩트. 학습 실패 아님.
- **앞으로 표준 지표 = loc + peak 상관 + rel(저force 제외) + 절대 rmse** (rel 단독 금지).
- 소재 서열(ecomesh 우세)은 절대 rmse·peak 상관으로도 유지되나, d10_rel 단독 수치는 재인용 금지.
