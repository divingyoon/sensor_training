# Fig.2 d5/d10 비교 — 6개 메트릭 수식·해석 방법 (방법 설명)

> 그림 `Fig2_d5d10_comparison.png` 의 6개 패널 각각이 **무엇을, 어떤 식으로** 계산했고,
> 그 값을 **어떻게 읽어 결론으로 연결하는지(해석 방법)** 까지 정리한다.
> 생성: `generate_2d_heatmap.py`(패널 1~3, Fig2B) + `generate_panelC_metrics.py`(패널 4~6, Fig2C)
> → CSV(`d5|d10/Fig2B_metrics.csv`, `Fig2C_metrics.csv`) → `generate_d5d10_comparison.py`(합본 그림).

각 패널은 **[수식] → [수식이 측정하는 물리량] → [값을 읽는 법] → [방향(↑/↓)의 의미] → [주의(함정)]** 순으로 읽는다.

## 0. 공통 정의 (먼저 알아야 할 것)

- **ΔS_i** (taxel i 의 신호) = `-(raw_i - baseline_i) / baseline_i × 100` [%]
  - baseline_i = 무접촉 초기 30 burst 평균. 부호: 압력 증가 = 양수.
- **|ΔS|** : 절댓값. 압입 시 폴리머 챔버가 인장/압축 어느 쪽으로 측정될지 불확실하므로
  "변화의 크기"로 비교한다(인접 taxel 의 **음수** 응답도 버리지 않음).
- **노이즈 기준** : 무접촉 std ≈ 0.01 %. → 절대 floor 는 그 50배인 **0.5 %**(배열) /
  σ 계산은 **2 %**(수용장 맵). 이보다 작은 값은 "반응"으로 세지 않는다.
- **센서 좌표** : 4×4, 간격 6.5 mm, 위치 ∈ {±9.75, ±3.25} mm.
- **격자** : 인덴터를 ±10 mm 를 1 mm 간격 스캔(21×21). 각 셀 = 그 위치의 **peak 압입** 1행.

### 두 종류의 "퍼짐"을 구분해서 읽어라 (핵심)
| | Fig2B (패널 1~3) | Fig2C (패널 4~6) |
|---|---|---|
| 보는 대상 | **taxel 1개(Skin10)** 의 ΔS | **한 번 압입의 16-taxel 벡터** |
| 변수 | 인덴터 위치 (x,y) 를 옮겨가며 | 한 점을 누른 순간 |
| 질문 | "이 taxel 이 **얼마나 넓은 영역의 압입**에 반응?" | "한 점을 누르면 **몇 개 taxel 로** 퍼지나?" |
| 의미 | 수용장(receptive field)의 **크기** | 배열 응답의 **분산/중첩** |

> **왜 둘 다 보나**: SR 은 "물리 taxel 사이"를 보간한다. 그러려면 (B) 각 taxel 수용장이
> 넓어 이웃 압입에도 반응하고, **동시에** (C) 한 점 압입이 여러 taxel 에 고르게 걸쳐야 한다.
> 두 그림이 같은 결론(넓게·고르게 = mesh 유리)을 다른 각도로 교차검증한다.

### 읽기 전 체크 (모든 패널 공통)
- **막대 패턴**: 빗금 = d5(⌀5mm, 국소), 채움 = d10(⌀10mm, taxel 간격보다 큼).
- **추세선**: 점선 d5 / 실선 d10 — 소재(eco20→eco50→mesh) 순 단조성 확인용.
- **무엇을 비교?**: 같은 인덴터 안에서 *소재 3개* 를 비교(소재 효과 분리). d5↔d10 는 같은 소재끼리 비교.

## 패널 1 — Mean receptive width (mm), per-taxel 중앙4 평균  [Fig2B · 수용장 크기]

**[수식]** 각 taxel t 의 수용장 맵 `g_t(x,y)`(= taxel t 의 ΔS 를 인덴터 위치 (x,y) 함수로) 에서
```
A_half(t) = (#{cells : g_t ≥ 0.5·peak_t}) × (1 mm)²    # 그 taxel 자기 peak 의 50% 면적
d_half(t) = 2 · sqrt(A_half(t) / π)                    # 등가 원 지름 [mm]
수용장폭 = mean_{t ∈ 중앙4(Skin6,7,10,11)} d_half(t)
```
**[측정 물리량]** "한 taxel 이 **자기 peak 의 절반 이상으로 반응하는 인덴터 위치 영역의 지름**" =
그 taxel 의 수용장 크기. 이를 **중앙 4개 taxel 에 대해 평균**한다(Skin10 하나만 쓰면 그 taxel
특이성이 섞임; 가장자리 taxel 은 수용장이 스캔 ±10mm 밖으로 잘려 제외).
> 주의: 이건 "한 점을 누르면 몇 개 taxel 이 반응하나(=Fig2C active·prop-σ)"와 **다른** 개념 —
> 변수가 *인덴터 위치*인 **수용장(receptive field)** 이다(둘은 transpose 관계).

**[읽는 법]** 막대 높이를 mm 로, **taxel 간격 6.5mm 점선과 비교**한다. d10 eco50 6.6·mesh 6.3 ≈ 6.5mm
→ 이웃 수용장이 겹침(overlap)=SR 보간 가능. eco20 5.07 < 6.5 → 간극(undersampling).

**[방향]** **클수록·6.5mm 에 가까울수록 이웃 수용장 중첩↑ = SR 유리.** eco20 이 d5·d10 모두 최저.

**[주의]**
- **Skin10 단일 → 중앙4 평균으로 교체(2026-06-25)**: Skin10 만 쓰면 그 taxel 의 flat-top 특이성으로
  d5 eco20 이 거꾸로 넓게 나오는 착시가 있었다. 평균하면 eco20 이 정상적으로 최저(d5 3.56, d10 5.07).
- peak 는 d10 에서 셋 다 ~100%(포화)라 폭(이 패널)·active·σ 로 소재를 구분한다.

## 패널 2 — Receptive-field σ (mm)  [Fig2B · 수용장 크기, 보조]

**[수식]** 임계 `τ = max(0.2·peak, 2%)`, `w = g (g≥τ) else 0`:
```
centroid (cx,cy) = Σ w·(x,y) / Σ w
σ = sqrt( Σ w·((x-cx)²+(y-cy)²) / Σ w )           [mm]
```
**[측정 물리량]** 수용장 맵의 **응답가중 RMS 반경** = "무게중심에서 평균 몇 mm 퍼져 있나".
패널 1(반치폭)과 같은 "크기"를 분포 전체의 2차모멘트로 본 보조 지표.

**[읽는 법]** 패널 1 과 같은 방향으로 움직이는지 본다(둘이 일치하면 신뢰↑).
d10: eco20 2.25 < eco50 2.55 ≈ mesh 2.50 → 패널 1 과 같은 순서.

**[방향]** **클수록 넓은 수용장.** 단, half-max(패널 1)보다 꼬리(약한 주변 응답)에 민감.

**[주의]**
- **이전 버전의 오류**: 임계 없이 *모든 양수 셀* 로 계산해, 저SNR 맵(d5 eco20, peak~20%)에서
  멀리 흩어진 노이즈가 거리를 부풀려 **eco20 이 거꾸로 가장 큼(3.91)** 이었다.
  → 임계 후 계산으로 수정 → eco20 d5 = **1.59(최소)**, 정상.
- "σ 가 크다"를 무조건 좋게 읽지 말 것: 노이즈로 커질 수 있어 **반드시 peak·half-max 와 같이** 본다.

## 패널 3 — Active cells (receptive)  [Fig2B · 수용장 크기, 셀 카운트]  *(그대로 유지)*

**[수식]**
```
active_cells = #{cells : g(x,y) > 0.2·peak}
```
**[측정 물리량]** taxel 이 "확실히(peak의 20% 이상)" 반응한 **인덴터 위치(격자 셀) 개수**.
넓이의 직접 카운트.

**[읽는 법]** 정수 개수. d10 eco20 43 → "21×21 격자 중 43곳의 압입에서 Skin10 이 반응".

**[방향]** **많을수록 수용장 넓음.** 패널 1·2 와 같은 방향이어야 정상(교차검증).
d10: 43 < 52 ≈ 54.

**[주의]** "peak 대비 20%" 상대 기준이라, peak 가 매우 작은(완전 무반응) 소재면 노이즈로 부풀 수 있음.
여기선 셋 다 peak 충분(>20%)이라 안전.

## 패널 4 — Total |ΔS| (%)  [Fig2C · 민감도]

**[수식]** 셀별 peak 압입에서
```
Total = Σ_{i=1..16} |ΔS_i|       (전 셀 평균)
```
**[측정 물리량]** 한 번 누를 때 **16개 taxel 이 만든 신호의 총합** = 소재의 **절대 민감도(세기)**.

**[읽는 법]** % 합. d10 eco50 114 → "강하게 반응", eco20 74 → "약함".

**[방향]** **클수록 민감.** 단 이건 "넓이"가 아니라 "세기" — SR 의 핵심(분산)이 아니라
**전제조건(신호가 충분히 큰가)**. mesh 가 eco50 급(112)이면 "민감도 손실 없음"의 근거.

**[주의]** Total 이 크다고 SR 에 유리한 건 아니다(eco50 은 세지만 분산은 mesh 보다 약함).
민감도(패널4)와 분산(패널5·6)을 **분리해서** 읽어야 한다 → mesh = "세기 유지 + 분산 최고".

## 패널 5 — Propagation σ (mm)  [Fig2C · 배열 분산]

**[수식]** `w_i = |ΔS_i| (|ΔS_i|>0.5%) else 0`, pos_i = taxel i 위치(mm):
```
centroid c = Σ_i w_i·pos_i / Σ_i w_i
σ_prop = sqrt( Σ_i w_i·|pos_i - c|² / Σ_i w_i )   [mm]
```
**[측정 물리량]** **한 점 압입의 신호가 16-taxel 물리 배열로 측방 전파한 RMS 폭**.
패널 1~3(한 taxel 의 수용장)과 달리 **여러 taxel 에 걸친 분산**을 본다.

**[읽는 법]** mm. 0 에 가까우면 "한 taxel 에만 집중", 크면 "여러 taxel 로 퍼짐".
d10: eco20 2.82 < eco50 3.61 ≈ mesh 3.58 → "eco50·mesh 가 이웃으로 넓게 전달, eco20 은 국소".

**[방향]** **클수록 수용장 중첩↑ = SR 가 보간할 입력이 풍부.** d10 에서 eco50≈mesh(동급), eco20 최저.

**[주의]**
- **이전 버전 퇴화**: 상대임계(0.15·peak)+부호 ΔS 였다. d5 처럼 1개 taxel 이 압도적이면
  그 1개만 남아 **σ_prop=0**(분산 없음으로 잘못 계산), 음수 이웃도 버렸다.
  → `|ΔS|`+절대 floor 0.5% 로 수정.
- **baseline drift 도 교정**: per-press local baseline(접촉 직전 기준) 적용 전에는 스캔 drift 가
  먼 taxel 을 띄워 d5 eco20 σ_prop 가 3.22 로 부풀려졌다. 교정 후 **0.61(최저)** 로 정상화.
  σ_prop·entropy 의 d10 mesh 우위(이전 5.56)도 drift 인공물이었고, 교정 후 eco50 과 동급(3.58).

## 패널 6 — Entropy H_norm  [Fig2C · 배열 분산의 균등성]

**[수식]** `p_i = w_i / Σ_j w_j` (패널5 와 같은 가중):
```
H_norm = -Σ_i p_i · ln(p_i) / ln(16)      ∈ [0,1]
```
**[측정 물리량]** 응답이 **얼마나 여러 taxel 에 고르게** 분산됐나.
0 = 한 taxel 독점, 1 = 16개 완전 균등.

**[읽는 법]** 0~1 무차원. d10: eco20 0.242 < eco50 0.349 ≈ mesh 0.340.
eco20 0.034(d5)·0.242(d10) → "특정 taxel 에 더 쏠림(국소)".

**[방향]** **클수록 균등 분산 = SR 가 보간하기 좋은 매끄러운 입력 분포.** d10 에서 eco50≈mesh, eco20 최저.

**[패널 5 와 차이 — 함께 읽는 법]**
- 패널5(σ_prop)는 **"얼마나 멀리"** 퍼졌나(거리). 패널6(H)은 **"얼마나 고르게"** 퍼졌나(균등성).
- d10 에서 eco50·mesh 는 σ_prop·H 둘 다 동급(넓고 고르게). mesh 의 단독 우위는 **active taxel 수**(패널3 계열).

**[주의]** 이전엔 ① 상대임계로 eco20 d5 가 **0 퇴화**, ② 전역 baseline drift 로 부풀림 → 절대 floor + per-press
local baseline 으로 교정(eco20 d5 0.034 정상화). 옛 "mesh 0.45 최고"(전역 baseline)는 drift 인공물.

## 종합 해석 (그림 → 결론으로 연결하는 법)

읽는 순서: **① 민감도 충분한가(패널4) → ② 수용장 넓은가(패널1~3) → ③ 분산이 넓고 고른가(패널5·6).**
SR 에 좋은 소재 = ①을 잃지 않으면서 ②③이 큰 것.

| 측면 | 보는 패널 | 결론 |
|---|---|---|
| 민감도(전제) | 4 Total\|ΔS\| | eco50(최고) ≈ mesh ≫ eco20 → mesh 민감도 유지 |
| 수용장 크기 | 1 half-max · 2 σ · 3 active | eco20 좁음 ≪ eco50·mesh 넓음 |
| 다중-taxel 관여 | 3 active · 5 prop-σ · 6 entropy | **d10 active = mesh 최고(6.9)**; σ·entropy 는 **eco50≈mesh 동급** |

→ **mesh20** = "eco50급 민감도 유지 + **가장 많은 taxel 활성화(active 최대)** + eco20 의 undersampling 회피" = SR 입력 최적. (C1)
  단, "mesh 가 σ·entropy 로 가장 넓고 고르게"는 **per-press local baseline 적용 후 성립 안 함**(eco50 와 동급, 이전엔 drift 부풀림).

> **d5 vs d10 읽기**: d5(국소)는 *소재 민감도 대비* 가 선명하지만 분산 신호가 약해 단독 판단 위험.
> d10(인덴터 > taxel 간격)이 여러 taxel 을 한 번에 덮어 **분산(SR) 주장을 가장 깨끗이** 보여준다.
> → **분산 결론은 d10 우선**, d5 는 민감도 대비·보조 확인용으로 읽는다.

## 수정 이력 (이번 리비전)
1. **Peak ΔS → Half-max receptive width**: d10 포화(100%)로 무의미 → 50% 기준 폭으로 교체(포화에 강건).
2. **Receptive σ 임계화**: 저SNR(d5 eco20) σ 부풀림 제거 → eco20 가 정상적으로 최소.
3. **배열 메트릭 `|ΔS|`+절대 floor(0.5%)**: propagation σ=0 / entropy=0 퇴화 해소, 인접 taxel 음수 응답 포함.
4. **per-press local baseline (2026-06-25)**: 압입 직전(접촉 전 z<13mm) per-taxel 평균을 baseline 으로 →
   스캔 drift 제거. d5 eco20 σ_prop 3.22→0.61 교정, d10 σ·entropy 의 mesh 우위(drift 인공물)가 사라져
   eco50 과 동급으로 정정. mesh 의 실제 우위는 active taxel 수로 수렴.
