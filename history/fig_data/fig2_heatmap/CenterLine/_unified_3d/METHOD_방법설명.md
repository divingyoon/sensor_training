# CenterLine 3D 시각화 — 공통 축 버전 (방법·의미 설명)

> 그림(PNG)은 지워도 되지만, **어떻게 만들었고 무슨 뜻인지**는 이 문서로 남긴다.
> 생성: `python generate_centerline_unified.py` → `_unified_3d/centerline_<소재>_<인덴터>.png` 6장.

---

## 1. 무엇을 만들었나
`combined_centerline.csv` 하나에서 **소재 3종 × 인덴터 2종 = 6개** 조합을 각각 개별 PNG로 그렸다.

| 소재 | 인덴터 |
|---|---|
| eco20 / eco50 / mesh20(=eco20+mesh) | d5(⌀5mm) / d10(⌀10mm) |

핵심: **6장 모두 x·y·z 축 범위, 시야각, 박스 비율을 똑같이 고정** → 나란히 놓으면 직접 비교 가능.
(기존 `visualize_3d_v3.py` 는 test 폴더마다 따로 그려 축이 제각각이라 비교가 안 됐다.)

---

## 2. 사용한 식 (신호 정의)

각 barometer(센서 i)의 **baseline 대비 변화율**을 z값으로 쓴다:

```
ΔS_i(t) = ( s_i(t) − baseline_i ) / baseline_i × 100      [단위: %]

baseline_i = 무접촉('unloaded') 구간에서 센서 i 의 raw 평균
```

- `s_i(t)` = 시각 t 에서 센서 i 의 raw ADC 값 (combined_centerline.csv 의 `s1`~`s16`).
- **부호 의미**: 인덴터가 누르면 공기 챔버가 압축되어 raw 가 **감소** → ΔS 가 **음수**.
  즉 **아래로 깊을수록(더 음수일수록) 접촉 응답이 강하다.** (기존 스크립트와 같은 부호.)
  - 참고: xy_1mm 쪽 Fig.2 는 부호를 뒤집어 `−(raw−base)/base×100` (양수)로 썼다. 여기 CenterLine 은
    원본 스크립트 관례를 유지해 **음수**가 강한 응답이다. (혼동 주의)

---

## 3. 세 축의 의미

| 축 | 의미 | 비고 |
|---|---|---|
| **x = Skin Sensor (1~16)** | 16개 barometer (4×4 배열) | 어느 센서가 반응하는지 = 공간 분포 |
| **y = Accumulated Time (s)** | 인덴터가 센터라인 중앙을 깊이 0→2mm 로 **반복 압입**한 누적 시간 | 압입/해제 사이클이 시간축으로 쌓임 |
| **z = Sensor Change ΔS (%)** | 위 식의 값 | 음수 골이 깊을수록 강한 접촉 |

> 이 데이터는 인덴터가 한 점(센터)에서 **깊이를 반복적으로** 넣었다 빼는 실험이다
> (x 위치는 거의 고정, z=깊이 0~2mm 반복). 그래서 시간축에 압입 골이 주기적으로 나타난다.

---

## 4. 공통으로 고정한 값 (전 그림 동일)

스크립트가 6개 조합 전체를 한 번 스캔해 전역 범위를 구하고 다음으로 고정한다:

- x 범위: **1 ~ 16**
- y 범위: **0 ~ 2400 s**  (모든 test 가 ~2346s 로 거의 동일)
- z 범위: **−70 ~ +15 %**  (전 그룹 ΔS 의 전역 최소~최대를 5% 단위로 라운드)
- 시야각: **elev=20°, azim=−60°**, 박스비율 `(1.5, 1.7, 0.85)` 동일
- 센서당 표시 포인트: 약 3000점으로 균일 다운샘플 (가독성)
- 센서 색: Group A(1–4) 파랑 / B(5–8) 주황 / C(9–12) 초록 / D(13–16) 빨강

→ 그림들 사이의 **차이는 전부 실제 소재·인덴터 차이**다 (축 차이가 아니라).

## 5. 그림을 읽는 법 (무슨 뜻인지)

- **인덴터 d5 vs d10**: d10(큰 인덴터)일수록 골이 더 깊고(=응답 큼) 더 많은 센서가 반응(넓음).
  d5(작은 인덴터)는 국소·약함.
- **소재 비교**:
  - **eco20**: 골이 얕다(예: d5 약 −25%). 신호 전달 약함 → 수용장 좁음(undersampling 경향).
  - **eco50**: 더 깊고 넓다.
  - **mesh20(ours)**: 골이 깊으면서(민감도 유지) **여러 센서로 넓게 퍼짐**(예: d10 약 −65%, 다중 그룹 반응).
    → mesh 가 압력을 측방으로 분산해 **수용장 중첩↑ + 민감도 유지** = SR 에 유리.

즉 이 그림 세트는 **"mesh 가 응답을 더 넓게·강하게 분산한다"(C1 주장)** 를 시간축 파형으로 보여준다.
