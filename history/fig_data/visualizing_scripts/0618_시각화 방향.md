# 1차 추천안

## 핵심 방향

목표는 단순히 “센서값이 크게 나온다”가 아니라, **EcoMesh가 ECO50처럼 압력 전파 범위가 넓으면서도, 위치 추론에 필요한 spatial gradient를 유지함**을 보여주는 것임.

따라서 시각화는 아래 3가지를 동시에 보여주는 구조가 적절함.

1. **압력 전파 범위가 넓음**
2. **센서 간 분포 차이가 충분히 존재함**
3. **SR / regression 학습에 유리한 입력 패턴을 형성함**

---

# 1. Centerline 실험 시각화

조건:

`y = 0 고정`, `x를 0.1 mm step으로 이동`, `z 동일 깊이`, `실험중 Fz 기록`

```

### color 값 추천

두 가지를 반드시 분리해서 보여주는 것이 좋음.

### 1) Absolute response

[

\Delta P_i(x) = P_i(x) - P_i^0

]

또는 힘 보정 시:

[

\frac{\Delta P_i(x)}{F_z}

]

이 그림은 **실제 전파 세기** 비교에 필요함.

### 2) Normalized response

[

\tilde{P}*i(x) =\frac{\Delta P_i(x)}{\max*{i,x} \Delta P_i(x)}]

이 그림은 **전파 패턴의 형태** 비교에 필요함.

### 해석 포인트

- ECO20: contact 주변 센서에만 응답이 집중될 가능성이 큼.
- ECO50: 더 넓은 범위의 센서가 반응하나, 지나치게 퍼지면 위치 분해능 저하 가능성이 있음.
- EcoMesh: ECO50 수준의 전파 범위를 보이면서, 센서별 gradient가 유지되면 가장 설득력 있음.

# 3. Propagation width 정량화

전파력이 좋다는 주장을 하려면 반드시 metric이 필요함.

## Metric 1. Effective number of activated sensors

각 x 위치에서 일정 threshold 이상 반응하는 센서 수를 계산함.

# [N_{\text{active}}(x)

\sum_i \mathbf{1}

\left(

\tilde{P}_i(x) > \tau

\right)

]

보통 (\tau = 0.1) 또는 (0.2) 사용 가능함.

이후 평균값:

# [\bar{N}_{\text{active}}

\frac{1}{N_x}

\sum_x N_{\text{active}}(x)

]

### 추천 그래프

```
Average Activated Sensors

ECO20        | ##
EcoMesh      | #####
ECO50        | ######
```

### 해석

- ECO20보다 EcoMesh의 (N_{\text{active}})가 커야 함.
- EcoMesh가 ECO50과 유사하면 “전파력 측면에서 ECO50에 근접”하다고 주장 가능함.

---

## Metric 2. Spatial spread / second moment

센서 위치를 (\mathbf{r}_i = [x_i, y_i]), 센서 응답을 (w_i)로 둘 때:

# [w_i(x)

\frac{\Delta P_i(x)}

{\sum_j \Delta P_j(x)}

]

전파 중심:

# [\boldsymbol{\mu}(x)

\sum_i w_i(x)\mathbf{r}_i

]

전파 폭:

# [\sigma_{\text{prop}}(x)

\sqrt{

\sum_i w_i(x)

\left|

\mathbf{r}_i - \boldsymbol{\mu}(x)

\right|^2

}

]

평균 전파 폭:

# [\bar{\sigma}_{\text{prop}}

\frac{1}{N_x}

\sum_x \sigma_{\text{prop}}(x)

]

### 추천 그래프

```
Propagation Width σprop

ECO20
EcoMesh
ECO50
```

### 해석

- ECO20은 (\sigma_{\text{prop}})가 작음.
- ECO50은 큼.
- EcoMesh가 ECO50과 유사한 (\sigma_{\text{prop}})를 보이면 전파력 비교에 강함.

---

## Metric 3. Response entropy

센서 응답이 몇 개 센서에 집중되는지, 전체로 퍼지는지 보여주는 지표임.

# [H(x)

- \sum_i w_i(x)\log w_i(x)

]

정규화하면:

# [H_{\text{norm}}(x)

\frac{H(x)}{\log N_s}

]

여기서 (N_s)는 센서 개수임.

### 해석

- 낮은 entropy: 한두 센서에만 응답 집중
- 높은 entropy: 여러 센서로 응답 분산
- EcoMesh가 ECO20보다 entropy가 높고 ECO50과 유사하면 전파성이 좋다고 주장 가능함.

단, entropy가 너무 높으면 모든 위치에서 비슷한 패턴이 되어 localization이 어려워질 수 있음. 따라서 entropy만 단독으로 쓰면 안 됨.

# 4. Centerpress 실험 시각화

조건:

((x,y)=(0,0)), 동일 깊이로 압입

이 실험은 **한 지점에서 눌렀을 때 압력이 센서 배열 전체로 어떻게 퍼지는지**를 보여주는 목적임.

## Figure C. 2D sensor response map

센서의 실제 위치를 기준으로 color map을 그림.

```
Center Press Response Map

ECO20          EcoMesh          ECO50

 [sensor map]   [sensor map]    [sensor map]
```

### 표시 방식

센서 위치가 격자라면 각 센서를 원 또는 사각형으로 표시하고, 색으로 (\Delta P_i/F_z)를 표현함.

```
S1   S2   S3

S4   C    S5

S6   S7   S8
```

중앙 C는 indentation point.

### 이 그림에서 보여야 하는 것

- ECO20: 중앙 근처 센서 위주로 반응
- EcoMesh: 주변 센서까지 반응 확산
- ECO50: 전체적으로 넓은 반응
- EcoMesh가 ECO50과 유사한 분포를 보이면 주장 강화 가능함

---

## Figure D. Radial attenuation plot

center press에서 각 센서의 거리 (r_i)를 계산하고, 응답을 거리 기준으로 plot함.

[

]

[

\quad \text{vs.} \quad

r_i

]

### 추천 형태

```

### 해석

- ECO20은 거리가 멀어지면 급격히 감소
- ECO50은 완만하게 감소
- EcoMesh가 ECO50처럼 완만한 감쇠 곡선을 보이면 전파력 주장이 명확해짐

# 5. SR 학습과 연결되는 시각화

단순히 전파력이 좋다는 것에서 끝나면 약함.

최종 주장은 **“EcoMesh가 학습 입력으로 더 좋은 tactile representation을 제공한다”**가 되어야 함.

따라서 아래 그림 중 하나는 넣는 것이 좋음.

---

## Figure E. Adjacent-position distinguishability

x가 0.1 mm씩 이동할 때 센서 벡터가 얼마나 구분되는지 계산함.

센서 응답 벡터:

# [\mathbf{s}(x)

[

\Delta P_1(x),

\Delta P_2(x),

...,

\Delta P_N(x)

]^T

]

인접 위치 간 차이:

# [D(x)

## \left|\mathbf{s}(x+\Delta x)

\mathbf{s}(x)

\right|_2

]

또는 cosine distance:

# [D_{\cos}(x)

1 -

\frac{

\mathbf{s}(x+\Delta x)^T\mathbf{s}(x)

}{

|\mathbf{s}(x+\Delta x)|

|\mathbf{s}(x)|

}

]

### 해석

- 너무 낮으면 위치가 바뀌어도 센서 패턴이 거의 같음 → 학습 불리
- 너무 높고 불연속적이면 noise-sensitive함
- 좋은 소재는 (D(x))가 충분히 크고 부드러워야 함

EcoMesh가 여기서 가장 좋은 결과를 보이면 “SR 학습에 유리”라는 주장이 강해짐.

---

## Figure F. Prediction error 비교

최종적으로 SR 학습까지 연결하려면, 가장 설득력 있는 그림은 이것임.

동일한 모델 구조로 각 소재 데이터를 학습시킨 뒤:

# [\hat{x}, \hat{y}, \hat{z}, \hat{F}_z

f_{\theta}(\mathbf{s})

]

각 material별 error를 비교함.

[

e_x = |\hat{x} - x|

]

[

e_z = |\hat{z} - z|

]

[

e_{F_z} = |\hat{F}_z - F_z|

]

### 추천 그래프

```
Position / Force Estimation Error

           x error     z error     Fz error
ECO20       ...
EcoMesh     ...
ECO50       ...
```

또는 box plot:

```
MAE of x estimation

ECO20     [ box ]
EcoMesh   [ box ]
ECO50     [ box ]
```

### 주장 구조

- ECO20은 응답 범위가 좁아 주변 위치 구분성이 낮음
- ECO50은 전파는 좋지만 너무 부드러워져 일부 위치 gradient가 약해질 수 있음
- EcoMesh는 넓은 전파와 위치별 차이를 동시에 제공하여 추론 오차가 낮음

이 결과가 나오면 가장 강한 결론이 됨.

# 권장 결론 문장

> EcoMesh exhibits pressure propagation comparable to ECO50 while maintaining distinguishable spatial response patterns across indentation positions. This produces richer and smoother tactile features than ECO20, which is beneficial for supervised regression of (x), (y), (z), and (F_z).
> 

한국어 발표용으로는 다음이 적절함.

> EcoMesh는 ECO20 대비 넓은 압력 전파 특성을 보이며, ECO50과 유사한 수준으로 주변 센서까지 응답을 전달함. 동시에 압입 위치 변화에 따른 센서 응답 패턴의 차이가 유지되므로, (x), (y), (z), (F_z) 추론을 위한 학습 입력으로 더 풍부한 tactile feature를 제공함.
>
