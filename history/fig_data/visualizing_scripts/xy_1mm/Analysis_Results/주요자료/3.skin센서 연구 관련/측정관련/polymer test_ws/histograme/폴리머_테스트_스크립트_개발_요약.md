# 폴리머 테스트 스크립트 개발 요약

## 1. 초기 요구사항: 모터 제어 기능 추가
- `polymer_test.py`에 `automated_calibration_cli.py`의 모터 제어 기능을 통합하도록 요청받았습니다.
- `MotorController` 클래스를 `polymer_test.py`에 추가하여, 시리얼 통신을 통해 모터를 제어하는 기본 기능을 구현했습니다.
- 사용자가 터미널에서 직접 모터를 제어할 수 있도록 인터랙티브 명령어 (`m [축] [스텝]`)를 추가했습니다.

## 2. 기능 개선: mm 단위 이동 및 Z축 전용 제어
- 모터 제어 단위를 스텝(step)에서 밀리미터(mm)로 변경하도록 요청받았습니다.
- `STEPS_PER_MM = 640` 상수를 적용하여 `move_mm` 메소드를 구현, mm 단위 제어가 가능하도록 수정했습니다.
- 실험에 Z축만 사용된다는 요구사항에 맞춰, 사용자 명령어를 `z [거리(mm)]`로 단순화했습니다.

## 3. 고급 기능 추가: 자동화 및 센서 확장
- **자동 원점 복귀:** 폴리머 센서 값이 0이 되면, 자동으로 원점으로 돌아오는 기능을 추가했습니다.
- **AFD50 센서 연동:**
    - `python-can`을 이용해 AFD50 센서의 Z축 힘(Fz) 데이터를 읽어오는 `Afd50Reader`를 구현하고 연동했습니다.
    - CSV 로그와 실시간 플롯에 AFD50 데이터를 함께 기록하고 표시하도록 수정했습니다.
- **실행 순서 변경:** 사용자가 첫 모터 이동 명령을 입력했을 때, 플롯 창이 표시되고 로깅이 시작되도록 로직을 변경했습니다.

## 최종 결과물
`polymer_test.py`는 폴리머 센서와 AFD50 센서 데이터를 동시에 수집, 플로팅, 로깅하는 기능을 갖추게 되었습니다. 또한, mm 단위의 정밀한 Z축 모터 제어 및 자동 원점 복귀 기능이 포함된 실험용 스크립트로 완성되었습니다.
