================================
폴리머 센서 테스트 스크립트 설명서
================================

이 프로젝트는 Z축 모터를 제어하여 폴리머 및 AFD50 센서 데이터를 수집하고, 수집된 데이터를 시각화하는 스크립트로 구성됩니다.

----------------
파일 설명
----------------

1. polymer_test.py
   - 용도: 모터를 제어하고, 연결된 센서들(폴리머, AFD50)의 데이터를 실시간으로 CSV 파일에 저장하는 메인 데이터 수집 스크립트입니다.
   - 주요 기능:
     * Z축 모터 정밀 제어 (mm 단위)
     * 폴리머 센서 및 AFD50 Force 센서 데이터 동시 수집 및 로깅
     * `max` 모드: 폴리머 센서 값이 0이 될 때까지 자동 전진 후 원점 복귀
     * `calibrate` 모드: 모터의 스텝-거리 변환 계수 자가 교정 기능
     * 자동 원점 복귀: 폴리머 센서 값이 0이 되면 자동으로 원점으로 복귀
   - 사용법:
     1. 터미널에서 `python polymer_test.py`를 실행합니다.
     2. 메뉴얼이 나타나면 원하는 명령을 입력합니다.
        * 숫자 (예: 10.5): 해당 거리(mm)만큼 이동 후 자동으로 원점 복귀합니다.
        * `max`: 센서 값이 0이 될 때까지 전진 후 원점 복귀합니다.
        * `calibrate`: 모터 이동 거리를 교정하는 절차를 시작합니다.
        * `exit`: 프로그램을 안전하게 종료합니다.

2. plot_csv_data.py
   - 용도: `polymer_test.py`가 생성한 로그(.csv) 파일을 분석하여 다양한 그래프를 생성하고 PNG 이미지 파일로 저장하는 스크립트입니다.
   - 주요 기능:
     * 가장 최근에 생성된 CSV 파일을 자동으로 찾아 분석합니다.
     * 원점 복귀 전까지의 유효 데이터만 사용하여 그래프를 생성합니다.
     * 총 5개의 그래프를 PNG 파일로 저장합니다:
       1. 시간 vs. 폴리머 센서
       2. 시간 vs. AFD50 Force
       3. Depth vs. 폴리머 센서
       4. 통합 그래프 (시간 vs. 모든 데이터, Y축 2개 사용)
       5. 3D 그래프 (시간 vs. Depth vs. Force)
   - 사용법:
     1. 터미널에서 `python plot_csv_data.py`를 실행합니다.
     2. 스크립트가 자동으로 최신 CSV 파일을 찾아 그래프들을 생성하고 같은 폴더에 PNG 파일로 저장합니다. 별도의 인자가 필요 없습니다.

----------------
작업 흐름
----------------

1. `polymer_test.py`를 실행하여 원하는 테스트(거리 입력 또는 `max` 모드)를 수행하고 `_log.csv` 파일을 생성합니다.
2. `plot_csv_data.py`를 실행하여 생성된 로그 파일을 기반으로 자동 생성된 그래프(.png)들을 확인합니다.
