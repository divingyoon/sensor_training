# 08_28_TEST CODE 개발

## 1. 최종 파일 요약

이 프로젝트는 데이터 수집용 스크립트, 데이터 분석용 스크립트, 그리고 전체 설명서로 구성됩니다.

-   **`polymer_test.py`**: 데이터 수집 및 모터 제어용 메인 스크립트입니다.
-   **`plot_csv_data.py`**: 수집된 CSV 로그 파일을 분석하여 그래프(PNG)로 저장하는 스크립트입니다.
-   **`README.txt`**: 프로젝트의 전반적인 설명과 각 스크립트의 사용법을 담은 설명서입니다.

## 2. WORKFLOW (작업 흐름)

이 프로젝트의 일반적인 작업 흐름은 데이터 수집, 분석, 문서 확인 순으로 진행됩니다.

### 단계 1: 데이터 수집 (`polymer_test.py`)

1.  **목적**: 실제 장비를 구동하여 테스트를 수행하고 모든 센서 및 모터 데이터를 CSV 파일로 기록합니다.
2.  **실행**: 터미널에서 `python polymer_test.py` 실행.
3.  **명령**:
    -   **거리 입력 (예: `20`)**: 입력한 거리(mm)만큼 Z축으로 하강했다가 자동으로 원점으로 복귀하는 기본 테스트를 수행합니다.
    -   **`max`**: 폴리머 센서 값이 0이 될 때까지 계속 하강한 후, 자동으로 원점으로 복귀하는 최대 압력 테스트를 수행합니다.
    -   **`calibrate`**: 모터의 이동 거리가 부정확할 때, 실제 측정값을 기반으로 `STEPS_PER_MM` 값을 교정하는 절차를 진행합니다.
4.  **결과**: 실행 날짜와 시간으로 된 `polymer_test_log_YYYYMMDD_HHMMSS.csv` 파일이 생성됩니다.

### 단계 2: 데이터 시각화 및 분석 (`plot_csv_data.py`)

1.  **목적**: `polymer_test.py`로 생성된 CSV 로그 파일을 읽어, 테스트 결과를 다양한 관점의 그래프로 시각화하고 이미지 파일로 저장합니다.
2.  **실행**: 터미널에서 `python plot_csv_data.py` 실행 (별도 인자 필요 없음).
3.  **동작**:
    -   스크립트가 현재 폴더 및 `data` 폴더에서 가장 최근에 만들어진 `.csv` 파일을 자동으로 찾습니다.
    -   원점 복귀 동작이 시작되기 전(센서 값이 0이 되기 전)까지의 데이터만 잘라내어 분석합니다.
    -   총 5개의 그래프를 PNG 파일로 자동 저장합니다.
      - `_time_vs_polymer.png`: 시간-폴리머 센서 관계
      - `_time_vs_force.png`: 시간-AFD50 힘 관계
      - `_depth_vs_polymer.png`: 깊이-폴리머 센서 관계
      - `_combined.png`: 시간-모든 데이터 통합 그래프 (Y축 2개 사용)
      - `_3d_plot.png`: 시간-깊이-힘 3D 관계
4.  **결과**: 5개의 `.png` 이미지 파일이 CSV 파일과 같은 위치에 생성됩니다.

### 단계 3: 추가 정보 확인 (`README.txt`)

1.  **목적**: 각 스크립트의 상세한 기능이나 필요한 라이브러리 등, 프로젝트에 대한 전반적인 내용이 기억나지 않을 때 확인합니다.
2.  **확인**: 텍스트 편집기로 `README.txt` 파일을 열어 내용을 읽습니다.
